	 	\textbf{Question}:
Let $\vec{A}$ be a $2\times 2$ real matrix with eigenvalues 1 and -1, and corresponding eigenvectors $\myvec{\sqrt{3}\\1}$ and $\myvec{2\\2}$.If $\vec{A}^{2021} = \myvec{a&b\\c&d}$,find a + b + c + d (round off to 2 decimal places).

\textbf{Solution:}
Required,
\begin{align}
		\myvec{1&1}\vec{A}^{2021}\myvec{1\\1} = a+b+c+d
\end{align}



As per Question,
\begin{align}
		\vec{A}\myvec{2\\2} = -1\myvec{2\\2}
\end{align}
\begin{align}
		\vec{A}\myvec{1\\1} = -1\myvec{1\\1}
\end{align}
\begin{align}
		\vec{A}^2\myvec{1\\1} = \myvec{1\\1}
\end{align}
So,
\begin{align}
		\vec{A}^{2021}\myvec{1\\1} = -1\myvec{1\\1}
\end{align}
\begin{align}
		\myvec{1&1}\vec{A}^{2021}\myvec{1\\1} = -1\myvec{1&1}\myvec{1\\1} = -2
\end{align}
